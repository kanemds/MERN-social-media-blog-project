REST API stands for Representational State Transfer Application Programming Interface.

It is an architectural style used in web services development for inter-system communication over the internet.

Resources (e.g., data objects or services) are identified by URLs, and interactions use standard HTTP methods (GET, POST, PUT, DELETE).
